# Cara setup backend

## buka terminal

## Jalankan peerintah di bawah

`cd be`

`composer install`

`cp .env.example .env`

`php artisan key:generate`

`php artisan migrate`

`php artisan serve`

##### **Pastikan api key openai sudah diisi di file .env saya tidak bisa memasukan api key di github**

# Cara setup frontend

## buka terminal baru

## Jalankan peerintah di bawah

`cd fe`

`npm install`

`npm run dev`
